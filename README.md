### You want running it in local? Here steps
1 - git clone
2 - npm i -g truffle
3 - install Ganache and runnit
4 - config ganache environemnt in truffle_config.js
5 - truffle migrate
6 - Install Metamask in the browser and config it for the ganache net.
7 - Ready for use
